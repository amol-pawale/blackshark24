## 🌟 Featured Projects

### 🛒 Commerce
![Stack](https://img.shields.io/badge/Stack-.NET_10_·_Clean_Architecture-512BD4?style=flat-square)
![Status](https://img.shields.io/badge/Status-In_Development-orange?style=flat-square)

**Production-grade e-commerce backend built on Clean Architecture**

A portfolio-grade backend showcasing enterprise architecture patterns end to end — domain-driven Clean Architecture layering, async messaging, caching, and API gateway routing, all containerized and observable.

**⚙️ How it's built:**
- 🧱 &nbsp;**Clean Architecture** — strict domain / application / infrastructure separation on **.NET 10**
- 🐘 &nbsp;**PostgreSQL + EF Core** — Code-First migrations as the persistence layer
- 🐇 &nbsp;**RabbitMQ** — async event-driven messaging between services
- ⚡ &nbsp;**Redis** — caching layer for high-throughput reads
- 🚪 &nbsp;**Kong API Gateway** — routing, rate limiting, and edge policy
- 📊 &nbsp;**OpenTelemetry + Grafana Cloud** — traces, metrics, and logs
- 🔄 &nbsp;**GitHub Actions CI/CD** — self-hosted runner with automated PR review
